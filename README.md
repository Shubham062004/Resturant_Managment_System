# 🔥 Oven Xpress - Restaurant Management System

Oven Xpress is an enterprise-grade, cloud-native Restaurant Management System (RMS) designed to unify FOH (Front-of-House) and BOH (Back-of-House) workflows.

This repository houses the complete production-ready project foundation, fully scaffolded with a modular feature-based architecture, dual-database capability, containerized orchestration, strict linting parameters, and robust CI/CD validation.

---

## 🏗️ Project Overview & Features
Oven Xpress implements a monorepo workspace layout.

### Core Features (Verified Implemented):
- **Authentication**: JWT-based auth, Role-Based Access Control (RBAC).
- **Core Operations**: Cart, Checkout, Order Management.
- **KDS (Kitchen Display System)**: WebSockets for real-time order states.
- **POS (Point of Sale)**: Hardware integrations and cashier operations.
- **Delivery Management**: Driver tracking and assignment.
- **Inventory & Reservations**: Stock alerts, Waitlist management.
- **AI & Analytics**: Sales forecasting and recommendation engines.
- **Multi-Tenant Super Admin**: Organization and branch hierarchy management.

---

## ⚙️ Tech Stack & Architecture

- **Frontend**: React (v18), TypeScript, Vite, Tailwind CSS (v3), React Router DOM (v6), Redux Toolkit, React Query (v5).
- **Backend**: Node.js, Express, TypeScript, PostgreSQL (Primary DB), MongoDB (Logs/Analytics), Prisma ORM, Mongoose ODM, Zod, JWT.
- **Infrastructure**: Multi-stage Docker, Nginx, GitHub Actions (CI/CD).
- **Security**: Helmet, CORS, Express Rate Limiting, Zod Validation.

## ⚡ Quick Start (Local Development)

### Docker Compose
```bash
docker compose up --build -d
```
- **Frontend**: http://localhost
- **Backend API**: http://localhost:5000/api/v1/health

### Native Run
1. Install dependencies: `npm install`
2. Sync DB: `npm run build --workspace=backend`
3. Start Dev: `npm run dev`
